# ADR-0008: Docs IA — Home for why, Architecture for how

## Status

Superseded

## Context

PR #100 added `what_is_lakeflow_framework` and strengthened operating-model / modelling-paradigm language for positioning and discoverability, plus SEO plumbing (`html_baseurl`, canonical/OG, sitemap, robots).

After the docs rebrand, **What is** and **Concepts** overlapped. An earlier decision moved “why” to Home and renamed Concepts → Architecture.

## Decision (superseded)

Originally: Home for why; Architecture for how; retire What is as a redirect.

**Superseded by:** restore **What is the Lakeflow Framework** under **Get Started**, keep Architecture as the structural deep-dive, and keep Home as the marketing landing (without absorbing Why content). The high-level architecture JPEG from the slide deck lives on the What is page under the opening paragraph.

## Consequences

- Get Started: What is → Quick Start
- Architecture remains the operating-model / DABs / specs deep dive
- SEO phrases remain on the restored What is page
