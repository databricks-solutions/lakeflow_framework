Features
========

Use this hub to explore the capabilities you can configure with the
Lakeflow Framework (**LFF**). Some sections cover framework-native
features, including metadata-driven specs, validation, Python
extensibility, and quarantine handling. Others cover Databricks Lakeflow
SDP product features that LFF exposes through the data flow spec, with
links back to Databricks documentation where useful.

Start with a category below, or jump to the :doc:`complete A–Z index <feature_a_z>`
when you already know the feature name.
