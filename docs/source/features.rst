Features
========

Some capabilities are **native to Lakeflow Framework** (metadata-driven specs,
configuration, Python extensibility, quarantine, and more). Others are
**Databricks / SDP product features** the framework exposes through the data
flow spec. **Sources and targets** sit in between — platform types, wired
through LFF.

Browse by category below, or jump to the :doc:`complete A–Z index <features_a_z>`
when you already know the name.
