Get Started
===========

Welcome to the Lakeflow Framework (LFF) docs.

New here? You can start with a short product overview, or jump into a 15-minute hands-on path.

**Product overview** — :doc:`/get-started/what-is-lakeflow-framework`

A short introduction to what the framework is, why teams use it, and the core concepts behind Framework and Pipeline Bundles. Best if you want orientation before you deploy.

**Hands-on path** — :doc:`/get-started/quick-start`

Deploy the Lakeflow Framework (LFF) and sample pipelines in your Databricks workspace in about **15–20 minutes**. Best if you want to see something running first.

----

**Two paths from here**

* **Guided path** — :doc:`/get-started/quick-start`, then deepen as you need:

  * Architecture in :doc:`/architecture/index`
  * Step through the sample pipelines in :doc:`/samples/index`
  * Practical implementations in :doc:`/build/patterns/index` and :doc:`/samples/index`
  * Build your first pipeline bundle - step by step in :doc:`/build/index`
  * Deploy via :doc:`/deploy/index` — read :doc:`/deploy/before-you-deploy` first for deploy order and ownership

* **Or straight into it** — jump to Architecture, Samples, Build, Deploy, or Features from the top navigation when you already know what you need
